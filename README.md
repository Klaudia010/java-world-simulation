# Java World Simulation

Academic Java project created as part of Object-Oriented Programming coursework during Data Engineering studies.

The project is a turn-based 2D virtual world simulator. Different organisms live on a board, perform actions, move, reproduce, collide with each other and interact according to their species-specific rules.

## General project idea

The goal of the assignment was to implement a virtual world using object-oriented programming principles.  
The world contains animals, plants and a human player. Each organism has its own strength, initiative, position and behavior. During each turn, organisms act in order based on initiative and age.

## Main features

- Turn-based world simulation
- Object-oriented class hierarchy for organisms
- Separate animal and plant behavior
- Species-specific collision and action rules
- Human player controlled by keyboard input
- Human special ability
- Graphical user interface
- Saving and loading game state

## Technologies

- Java
- Object-Oriented Programming
- Java GUI library
